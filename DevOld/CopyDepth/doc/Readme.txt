=========
CopyDepth
=========

----
コマンド

CopyDepth.exe コピー元ディレクトリ 深さ

CopyDepth.exe コピー元ディレクトリ - コピーしないディレクトリ...


----
実行例

CopyDepth.exe C:\home\HPGame 2

CopyDepth.exe C:\home\HPGame - C:\home\HPGame\Sword\Storehouse

CopyDepth.exe C:\Data\ABC - C:\Data\ABC\SubDir-1 C:\Data\ABC\SubDir-2\A\001 C:\Data\ABC\SubDir-2\B\002


----
捕捉

コピー先ディレクトリは C:\1 〜 999 のいずれか。
終了時、勝手にフォルダを開く。

深さ
	0 ... コピーしない。(ツリー情報ファイルの出力のみ)
	1 ... コピー元の直下にあるファイルのみコピー
	2 ... コピー元の直下にあるファイル・ディレクトリと
		コピー元の直下にあるディレクトリの直下にあるファイルのみコピー
	3 ... コピー元の直下にあるファイル・ディレクトリと
		コピー元の直下にあるファイル・ディレクトリと
		コピー元の直下にあるディレクトリの直下にあるディレクトリの直下にあるファイルのみコピー


「コピーしないディレクトリ」に相対パスを指定した場合、カレントディレクトリからの相対パスであることに注意すること。
