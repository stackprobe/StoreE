====================
GitHubDownloadFilter
====================

----
説明

GitHub-にコミット・プッシュしてダウンロードした Dev を元に戻す。


----
処理内容

・改行コードを元に戻す。
・日本語を含むファイル・フォルダ名を元に戻す。
・圧縮されたマップデータの展開


----
コマンド

GitHubDownloadFilter.exe 入力リポジトリ-DIR

	入力リポジトリ-DIR ... リポジトリのページ ⇒ Code ⇒ Download ZIP
		からダウンロードしたアーカイブを展開したディレクトリ


	出力先は C:\1, C:\2, C:\3...
