==========
RingCipher
==========

----
コマンド

RingCipher.exe [/M | /F] (/K 鍵ファイル | /P パスワード) (/E | /D) (入出力ファイル * | 入力ファイル 出力ファイル)

	/M ... 強制的にファイル処理をメモリ上で行う。
	/F ... 強制的にファイル処理をメモリ上で行わない。


----
暗号化されたファイルの推奨拡張子

	.enc

例：

	CompressedFile.cmp.gz.enc


----
補足

入力ファイルサイズが 200,000,000 バイト以下のときはファイル処理をメモリ上で行う。

入力ファイルの破損や鍵・パスワードの不一致により処理に失敗したとき
出力ファイルは削除されることに注意すること。
入出力ファイルを指定した場合、そのファイルは消滅する。


----
鍵ファイル

ファイル形式
	テキスト

文字コード
	US-ASCII

改行コード
	CR-LF または LF

書式
	各行は偶数桁の16進数
	ファイル全体で 32 桁以上の 16 の倍数桁 (128 bit 以上の 64 の倍数 bit, 16 バイト以上の 8 の倍数バイト)
	でなければならない。
	空行は無視する。
	各行の行頭・行末の空白は無視する。
	半角セミコロンで始まる行はコメントとして無視する。
