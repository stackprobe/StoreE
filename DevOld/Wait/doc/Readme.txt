====
Wait
====

----
説明

TIMEOUT n コマンドのように n 秒間待ちます。その間キー入力も受け付けます。


----
コマンド

Wait.exe WAIT-SEC

	WAIT-SEC ... 初期の残り時間を示す秒数
	             指定できる範囲：0 〜 999999


実行例

	Wait.exe 60
	Wait.exe 180
	Wait.exe 3600


----
キー入力

キー               結果
-----------------------------------------------
エスケープキー   直ちに終了する。エラーレベル 1
エンターキー     直ちに終了する。エラーレベル 0
+ キー           残り時間プラス 1 分
- キー           残り時間マイナス 1 分
* キー           残り時間プラス 1 時間
/ キー           残り時間を残り 3 秒にする。


----
その他

キー入力により変更された残り時間は 3 〜 999999 sec の範囲になるように矯正されます。

タイムアウト終了時のエラーレベルは 0 です。

エラー終了時のエラーレベルは 1 です。
